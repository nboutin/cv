# CV

## How-To with Windows

### Install MiKTeX

https://miktex.org/howto/install-miktex

Allow on-the-fly package update

## How-To with Linux

### Install Latex

```shell
sudo apt install texlive-full
```

## Generate

```console
./gen.sh cv
./gen.sh full
./gen.sh all
```
